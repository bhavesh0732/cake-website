# cake-website

A responsive and visually appealing Cake Shop website built using HTML, CSS, and JavaScript. This project showcases a simple online bakery UI where users can browse cakes and view details.
🚀 Features

- 🍰 Attractive homepage UI
- 🧁 Product (cake) showcase section
- 📱 Fully responsive design (mobile + desktop)
- 🛒 Add to cart UI (basic frontend simulation)
- 🎨 Smooth and modern styling
- ⚡ Interactive elements using JavaScript

Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
